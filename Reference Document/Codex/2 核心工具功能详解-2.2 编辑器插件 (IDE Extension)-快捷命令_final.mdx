# Codex IDE 扩展斜杠命令

斜杠命令让您无需离开聊天输入即可控制 Codex。使用它们检查状态、在本地和云端模式之间切换，或发送反馈。

<AdPlaceholder />

## 使用斜杠命令

1. 在 Codex 聊天输入中，键入 `/`。
2. 从列表中选择一个命令，或继续键入以过滤（例如 `/status`）。
3. 按 **Enter**。

## 可用的斜杠命令

| 斜杠命令 | 描述 |
| -------- | ---- |
| `/auto-context` | 打开或关闭自动上下文以自动包含最近的文件和 IDE 上下文。 |
| `/cloud` | 切换到云端模式以远程运行任务（需要云端访问）。 |
| `/cloud-environment` | 选择要使用的云端环境（仅在云端模式下可用）。 |
| `/feedback` | 打开反馈对话框以提交反馈，可选择包括日志。 |
| `/local` | 切换到本地模式以在工作区中运行任务。 |
| `/review` | 启动代码审查模式以审查未提交的更改或与基础分支比较。 |
| `/status` | 显示线程 ID、上下文使用情况和速率限制。 |
