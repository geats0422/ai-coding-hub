## 斜杠命令

斜杠命令让您无需离开线程编辑器即可控制 Codex。可用命令因您的环境和访问权限而异。

### 使用斜杠命令

1. 在线程编辑器中，键入 `/`。
2. 从列表中选择一个命令，或继续键入以过滤（例如 `/status`）。

您还可以通过在线程编辑器中键入 `$` 来显式调用技能。请参阅[技能](https://developers.openai.com/codex/skills)。

启用的技能也会出现在斜杠命令列表中（例如 `/imagegen`）。

### 可用的斜杠命令

| 斜杠命令 | 描述 |
| -------- | ---- |
| `/feedback` | 打开反馈对话框以提交反馈，可选择包括日志。 |
| `/mcp` | 打开 MCP 状态以查看连接的服务器。 |
| `/plan-mode` | 切换多步规划的计划模式。 |
| `/review` | 启动代码审查模式以审查未提交的更改或与基础分支比较。 |
| `/status` | 显示线程 ID、上下文使用情况和速率限制。 |
