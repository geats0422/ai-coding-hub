## Overview

The System Prompt Override feature allows advanced users to customize or replace
the default system prompt used by Gemini CLI. This enables fine-tuned control over
the AI's behavior and responses.

## Usage

To override the system prompt, you can:

1. Use command-line flags
2. Configure through settings.json
3. Set environment variables

### Command-line usage

```bash
gemini --system-prompt "Your custom system prompt here"
```

### Configuration

Add custom system prompts to your `settings.json`:

```json
{
  "systemPrompt": {
    "custom": "Your custom system prompt...",
    "enabled": true
  }
}
```

## Considerations

- Custom system prompts may significantly alter AI behavior
- Test thoroughly before production use
- Be aware of potential security implications
- Consider the impact on tool usage and responses

## Best practices

- Start with small modifications
- Test in a safe environment
- Document your custom prompts
- Review changes regularly
