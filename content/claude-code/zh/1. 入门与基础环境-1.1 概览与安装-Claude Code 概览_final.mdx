# Claude Code 终极指南：从零开始使用 AI 编程助手

Claude Code 是 Anthropic 开发的代理编码工具，它位于您的终端中，帮助您比以往任何时候都更快地将想法转化为代码。在几分钟内，您就可以安装、配置并开始使用这个强大的 AI 助手。

<AdPlaceholder />

## 前置条件

在开始使用 Claude Code 之前，请确保您已准备好以下内容：

- **操作系统**: macOS 13.0+、Ubuntu 20.04+/Debian 10+ 或 Windows 10+（带 WSL 1、WSL 2 或 Git for Windows）
- **硬件**: 4 GB+ RAM
- **网络连接**: 稳定的互联网连接
- **Shell 环境**: Bash 或 Zsh（推荐）
- **账户**: [Claude 订阅](https://claude.com/pricing)（Pro、Max、Teams 或 Enterprise）或 [Claude Console](https://console.anthropic.com/) 账户

## 步骤 1: 安装 Claude Code

Claude Code 的安装非常简单。您有多种安装方式可供选择。

### 步骤 1.1：使用本机安装（推荐）

本机安装是最简单的方式，支持自动更新。

**macOS、Linux、WSL:**

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows PowerShell:**

```powershell
irm https://claude.ai/install.ps1 | iex
```

**Windows CMD:**

```batch
curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
```

<ProTip>
本机安装会在后台自动更新到最新版本，确保您始终拥有最新的功能和安全修复。
</ProTip>

### 步骤 1.2：使用 Homebrew 安装

```bash
brew install --cask claude-code
```

<Note>
Homebrew 安装不会自动更新。您需要定期运行 `brew upgrade claude-code` 来获取最新功能和安全修复。
</Note>

### 步骤 1.3：使用 WinGet 安装

```powershell
winget install Anthropic.ClaudeCode
```

<Note>
WinGet 安装不会自动更新。您需要定期运行 `winget upgrade Anthropic.ClaudeCode`。
</Note>

### 步骤 1.4：验证安装

安装完成后，运行以下命令验证安装：

```bash
claude doctor
```

## 步骤 2: 登录您的账户

首次使用时，系统会提示您登录。您可以使用以下任一账户类型：

- **Claude 订阅**（推荐）：订阅 Claude 的 [Pro 或 Max 计划](https://claude.ai/pricing)，在一个地方管理您的账户
- **Claude Console**：通过 [Claude Console](https://console.anthropic.com/) 连接，需要活跃的账单

登录命令：

```bash
claude
# 首次使用时系统会提示您登录
```

您也可以手动登录：

```bash
/login
```

## 步骤 3: 启动您的第一个会话

安装并登录后，导航到您的项目目录并启动 Claude Code：

```bash
cd your-project
claude
```

您将看到 Claude Code 的欢迎界面，其中包含您的会话信息、最近的对话和最新更新。

### 步骤 3.1：询问关于您的项目

开始探索您的代码库：

```bash
# 在 Claude Code 中输入
what does this project do?
```

```bash
# 或者输入
explain the project structure
```

```bash
# 或者询问技术栈
what technologies does this project use?
```

### 步骤 3.2：让 Claude 修改代码

描述您想要的功能，Claude Code 将自动规划、编写代码并进行修改：

```bash
# 示例
add input validation to the user registration form
fix the authentication bug
```

<Note>
Claude Code 在修改文件之前始终会请求您的批准，让您可以控制每一个更改。
</Note>

## Claude Code 可以为您做什么

### 从描述构建功能

用纯英文告诉 Claude 您想构建什么：

- **快速示例**："add a login form with email validation"
- **详细描述**："create a REST API endpoint that accepts POST requests at /api/users with email validation, uses standard error response format"

### 调试和修复问题

粘贴错误消息或描述问题，Claude Code 将分析代码库、识别根本原因并实施修复：

```bash
# 示例
there's a bug where users see a blank screen after login - fix it
TypeError: 'NoneType' object is not subscriptable in my code
```

### 导航任何代码库

Claude Code 维护对整个项目结构的理解，可以回答有关团队代码库的任何问题：

```bash
# 示例问题
where is the main entry point?
how does the authentication flow work?
what files handle database connections?
```

### 自动化繁琐任务

让 Claude 处理重复性工作，释放您的时间：

```bash
# 示例
fix all lint issues
resolve merge conflicts
write release notes
```

<ProTip>
Claude 可以直接编辑文件、运行命令和创建提交，非常适合这些自动化任务。
</ProTip>

## 常见问题排查

### 安装问题

如果安装失败：

- 检查是否满足系统要求
- 确保网络连接正常
- 对于 Homebrew 安装，确保 Homebrew 已更新
- 运行 `claude doctor` 检查安装类型和版本

### 权限错误

如果 Claude 拒绝某个操作，请检查您的权限配置。

### 登录问题

如果登录失败，请验证：
- 账户类型是否正确
- 网络连接是否正常
- 订阅或 Console 账户是否有效

Claude Code 是一个强大的 AI 编程助手，可以显著提升您的开发效率。通过按照以上步骤，您可以在几分钟内完成设置并开始使用它的强大功能。

从简单的代码修复到复杂的架构重构，Claude Code 都能帮助您更快地完成任务。它的 Unix 哲学设计让您可以将其集成到现有的工作流中，实现真正的自动化编程。
