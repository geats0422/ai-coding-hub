# Claude Code 快速入门指南：5分钟学会 AI 编程助手

欢迎使用 Claude Code！这个强大的 AI 编程助手将帮助您更快地编码、调试和维护项目。本指南将带您在几分钟内从零开始掌握 Claude Code 的核心功能。

<AdPlaceholder />

## 前置条件

在开始使用 Claude Code 之前，请确保您具备以下条件：

- **终端环境**：打开的终端或命令提示符
- **代码项目**：一个要处理的代码项目目录
- **账户**：有效的 [Claude.ai](https://claude.ai) 订阅账户或 [Claude Console](https://console.anthropic.com/) API 账户
- **网络连接**：稳定的互联网连接（用于身份验证和 AI 交互）

## 步骤 1：安装 Claude Code

### 步骤 1.1：选择安装方式

Claude Code 提供多种安装方式以适应不同的工作环境。

**选项 A：本机安装（推荐）**

**macOS、Linux、WSL：**
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows PowerShell：**
```powershell
irm https://claude.ai/install.ps1 | iex
```

**Windows CMD：**
```batch
curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
```

<ProTip>
本机安装会在后台自动更新到最新版本，确保您始终拥有最新的功能和安全修复。
</ProTip>

**选项 B：Homebrew 安装（macOS）**

```bash
brew install --cask claude-code
```

<Note>
Homebrew 安装不会自动更新，需要定期运行 `brew upgrade claude-code`。
</Note>

**选项 C：WinGet 安装（Windows）**

```powershell
winget install Anthropic.ClaudeCode
```

<Note>
WinGet 安装不会自动更新，需要定期运行 `winget upgrade Anthropic.ClaudeCode`。
</Note>

### 步骤 1.2：验证安装

安装完成后，打开新的终端窗口并运行：

```bash
claude --version
```

如果显示版本号，说明安装成功。

## 步骤 2：登录您的账户

### 步骤 2.1：启动登录流程

运行 `claude` 命令启动交互式会话：

```bash
claude
```

首次使用时，系统会自动提示您登录。

### 步骤 2.2：选择账户类型

您可以使用以下任一账户类型登录：

- **Claude.ai 订阅账户**（推荐）：订阅 Pro、Max、Teams 或 Enterprise 计划
- **Claude Console 账户**：使用预付费额度的 API 访问

### 步骤 2.3：完成身份验证

按照屏幕提示完成登录流程。登录后，凭证将存储在您的系统上，无需再次登录。

<ProTip>
如果使用 Claude Console 账户，系统会自动为您创建一个名为"Claude Code"的工作区，用于集中成本跟踪。
</ProTip>

## 步骤 3：启动您的第一个会话

### 步骤 3.1：进入项目目录

导航到您要工作的项目目录：

```bash
cd /path/to/your/project
```

### 步骤 3.2：启动 Claude Code

运行 claude 命令启动会话：

```bash
claude
```

您将看到欢迎屏幕，包含会话信息、最近对话和最新更新。

<ProTip>
确保您的项目是 Git 仓库，这样 Claude Code 可以更好地理解项目结构。
</ProTip>

### 步骤 3.3：查看帮助信息

输入以下命令查看可用命令：

```bash
/help
```

## 步骤 4：提出您的第一个问题

### 步骤 4.1：了解您的项目

让 Claude 分析您的代码库：

```bash

```

```bash

```

```bash

```

```bash

```

### 步骤 4.2：了解 Claude Code 自身

Claude Code 可以回答有关其自身功能的问题：

```bash
> what can Claude Code do?
```

```bash
> how do I use slash commands in Claude Code?
```

```bash
> can Claude Code work with Docker?
```

<Note>
Claude Code 根据需要自动读取您的文件，无需手动添加上下文。
</Note>

## 步骤 5：进行您的第一次代码更改

### 步骤 5.1：提出修改请求

尝试一个简单的任务：

```bash
> add a hello world function to the main file
```

### 步骤 5.2：理解修改流程

Claude Code 将执行以下步骤：

1. **定位文件**：找到合适的文件进行修改
2. **展示更改**：向您显示建议的代码更改
3. **请求批准**：等待您的确认
4. **执行修改**：获得批准后进行编辑

<Note>
Claude Code 在修改文件前始终请求权限。您可以批准单个更改或为会话启用"全部接受"模式。
</Note>

<ProTip>
对于简单任务，可以使用"全部接受"模式加快流程。
</ProTip>

## 常用命令速查表

| 命令 | 功能 | 示例 |
|------|------|------|
| `claude` | 启动交互模式 | `claude` |
| `claude "task"` | 运行一次性任务 | `claude "fix the build error"` |
| `claude -p "query"` | 运行查询后退出 | `claude -p "explain this function"` |
| `claude -c` | 继续最近对话 | `claude -c` |
| `claude -r` | 恢复之前对话 | `claude -r` |
| `claude commit` | 创建 Git 提交 | `claude commit` |
| `/clear` | 清除对话历史 | `> /clear` |
| `/help` | 显示帮助信息 | `> /help` |
| `exit` 或 Ctrl+C | 退出 Claude Code | `> exit` |

## 专业提示

### 提示 1：对您的请求要具体

**不要这样说：**
```bash
> fix the bug
```

**尝试这样说：**
```bash
> fix the login bug where users see a blank screen after entering wrong credentials
```

<ProTip>
越具体的描述，Claude 的回答就越准确。提供错误消息、预期行为和实际行为的对比。
</ProTip>

### 提示 2：使用分步说明

将复杂任务分解为多个步骤：

```bash
> 1. create a new database table for user profiles
```

```bash
> 2. create an API endpoint to get and update user profiles
```

```bash
> 3. build a webpage that allows users to see and edit their information
```

<ProTip>
分步执行可以让 Claude 更好地理解上下文，每个步骤都能基于前一步的结果。
</ProTip>

### 提示 3：让 Claude 先探索

在进行更改前，让 Claude 了解您的代码：

```bash
> analyze the database schema
```

```bash
> build a dashboard showing products that are most frequently returned by our UK customers
```

<ProTip>
Claude 会自动建立对代码库的理解，这有助于后续的修改更加准确。
</ProTip>

### 提示 4：使用快捷方式节省时间

- 按 `?` 查看所有可用的键盘快捷方式
- 使用 Tab 进行命令补全
- 按 ↑ 查看命令历史
- 输入 `/` 查看所有斜杠命令

### 提示 5：善用斜杠命令

Claude Code 提供多种斜杠命令：
- `/help`：获取帮助信息
- `/compact`：压缩对话历史
- `/clear`：清除当前对话
- `/config`：打开配置菜单
