# Codex IDE 扩展斜杠命令

斜线命令让您无需离开聊天输入即可控制Codex。使用它们来检查状态、在本地模式和云模式之间切换或发送反馈。

<AdPlaceholder />

## 使用斜杠命令

1. 在Codex 聊天输入中，输入`/`。
2. 从列表中选择一个命令，或继续键入进行过滤（例如，`/status`）。
3. 按**输入**。

## 可用的斜杠命令|斜线命令 |描述 |
| -------- | ---- |
| `/auto-context` |打开或关闭自动上下文以自动包含最近的文件和 IDE 上下文。 |
| `/cloud` |切换到云模式以远程运行任务（需要云访问）。 |
| `/cloud-environment` |选择要使用的云环境（仅在云模式下可用）。 |
| `/feedback` |打开反馈对话框以提交反馈，可以选择包括日志。 |
| `/local` |切换到本地模式以运行工作区中的任务。 |
| `/review` |启动代码审查模式以审查未提交的更改或与基础分支进行比较。 |
| `/status` |显示线程 ID、上下文使用情况和速率限制。 |
