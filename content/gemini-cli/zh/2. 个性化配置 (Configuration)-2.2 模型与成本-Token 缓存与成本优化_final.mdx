# 令牌缓存和成本优化

Gemini CLI 在使用时通过令牌缓存自动优化 API 成本
API 密钥身份验证（Gemini API 密钥或 Vertex AI）。这个功能重用了
以前的系统指令和上下文以减少令牌数量
在后续请求中进行处理。

<AdPlaceholder />

**令牌缓存可用于：**

- API 密钥用户（Gemini API 密钥）
- Vertex AI 用户（具有项目和位置设置）

**令牌缓存不适用于：**

- OAuth 用户（Google 个人/企业帐户） - Code Assist API 可以
  目前不支持缓存内容创建

您可以使用`/stats`查看您的令牌使用情况和缓存的令牌节省
命令。当缓存的令牌可用时，它们将显示在统计信息中
输出。
