# Gemini CLI 扩展

Gemini CLI扩展包提示、MCP服务器、自定义命令、挂钩、
将子代理和代理技能转换为熟悉且用户友好的格式。与
扩展，您可以扩展 Gemini CLI 的功能并共享这些功能
与他人的能力。它们的设计易于安装并且
可分享。

要查看扩展示例，您可以浏览图库
[Gemini CLI extensions](https://geminicli.com/extensions/browse/)。

<AdPlaceholder />

## 管理扩展

您可以使用交互式验证已安装的扩展及其状态
命令：
```
bas
h
/extensions list
```
或在非交互模式下：
```
bas
h
gemini extensions list
```
## 安装

要安装真正的扩展，您可以使用 `extensions install` 命令和
非交互模式下的 GitHub 存储库 URL。例如：
```
bas
h
gemini extensions install https://github.com/gemini-cli-extensions/workspace
```
